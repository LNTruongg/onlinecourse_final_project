# Online Course Final Project

This is a Django-based Online Course web application developed as the final project assignment.

## Features

- Django admin site
- Course and lesson management
- Question and choice models
- Exam submission system
- Exam result evaluation
- Bootstrap course detail page

## Technologies Used

- Python
- Django
- Bootstrap 5
- SQLite3

## Project Structure

- `models.py` — Contains Course, Lesson, Question, Choice, and Submission models
- `views.py` — Handles course details, exam submission, and exam results
- `urls.py` — URL routing
- `admin.py` — Django admin configurations
- `templates/` — HTML templates using Bootstrap
